## Selection Criteria (all candidates must satisfy)

1. **Accepts pseudonymous-but-identified BO.** Counsel must accept KYC where the BO is identified to counsel only (not public) and may use a long-term pseudonym for public-facing records.
2. **Crypto-specialization.** Real prior experience with crypto / DLT / DeFi / token projects. Not generic corporate practice.
3. **Foundation-style governance experience.** For CH/LI: experience setting up Stiftung. For BVI: experience with BC + nominee directors.
4. **Banking introductions available.** Counsel can introduce to Sygnum / SEBA / Bank Frick / equivalent.
5. **English working language.** All correspondence in English. German / French / Spanish as secondary, not primary.
6. **Reasonable scoping call response time.** Counsel responds to scoping inquiry within 2 weeks.
7. **Engagement cost in range.** Setup phase: USD 15K–80K. Annual retainer: USD 10K–40K.

## Counsel-Acceptance Hierarchy

When scoping calls return, evaluate counsel responses against these tiers:

- **Tier 1 — full acceptance:** counsel accepts pseudonymous BO with documented nominee structure, crypto-specialized, banking introductions, scoping call scheduled within 2 weeks.
- **Tier 2 — conditional acceptance:** counsel accepts pseudonymous BO but adds documentation requirements beyond standard nominee (e.g., physical meeting required, identity-verification on long-term-residency document). Acceptable if documentation requirements are operationally feasible.
- **Tier 3 — reject:** counsel requires real-name disclosure. Out of scope per CONTEXT.md D-05.
