Plan ships with a single `deploy/docker-compose.yml` + 1-page operator guide enabling external operators to deploy in <1 hour (success criterion #3).

Purpose: Deliver the on-protocol substrate the frontend (Plan 02) and distribution stack (Plan 03) build against. Signet-only; mainnet exposure is gated on Phase 3 audit per CONTEXT.md item 9.
